# 読書記録サイト (dokusho)

## プロジェクト概要
読んだ本の内容を自分なりに理解し、他者に説明できる形でまとめて公開するサイト。

## ファイル構造
```
.claude/
├── CLAUDE.md              # このファイル（索引）
├── docs/
│   ├── WEB_TECH.md        # Web技術メモ（未作成）
│   └── books/
│       ├── INDEX.md       # 本の一覧
│       └── [タイトル].md  # 各本の記録
└── settings.local.json
```

## 参照ファイル
- @docs/books/INDEX.md - 本の一覧・索引
- @docs/books/修身.md - 修身（社長から借りた本）
